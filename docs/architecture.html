  • touch $DTACH_SIGNAL_DIR/$DTACH_SOCKET_INDEX.signal

⚙ VS Code Extension

  • fs.watch + 10s polling fallback
  • Signal → in-memory Map<index, Signal>
  • Status bar: ● N awaiting
  • Terminal switch → clears signal + deletes file
  • Status bar click → quick pick → focus terminal
  • Terminal close → clears signal + notifies watcher
